    O PokéRanking é uma aplicação que fornece um sistema de ranqueamento e classificação de pokémons de todas as gerações!

    Com ele você pode rankear os pokémons aplicando filtros, fazer comparações entre criaturas diversas, além de visualizar informações e detalhes individuais dos pokémons e fazer uma simulação de equipe!

    O projeto foi criado para a disciplina de Métodos de Desenvolvimento de Software do curso de Engenharia de Software da Universidade de Brasília (UnB), Campus Gama. Para mais informações, consulte a aba Documentação.
